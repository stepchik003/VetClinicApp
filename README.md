# Приложение "Ветеринарная клиника"

## Описание проекта
Мобильное приложение для ветеринарной клиники, разработанное на Android на языке программирования Kotlin.

## Функциональность
* **Новый пациент** - создание и редактирование карточек пациентов (животных)
* **Запись ко врачу** - создание, просмотр и редактирование записей на прием
* **Медицинские карточки** - просмотр данных и истории визитов для каждого питомца

## Архитектура и технологии
* __Язык:__ Kotlin

* __Архитектура:__ MVVM

* __UI:__ Jetpack Compose

* __Навигация:__ Jetpack Navigation

* __База данных:__ Room

* __DI:__ Hilt

* __Асинхронность:__ Kotlin Coroutines + Flow

## Схема базы данных
### Сущности

#### Patient (Пациент)
* ```id``` - идентификатор (Long, Primary Key)
* ```name``` - имя животного (String)
* ```species``` - вид животного (String)
* ```breed``` - порода (String, nullable)
* ```age``` - возраст (Int, nullable)
* ```weight``` - вес (Double, nullable)
* ```createdAt``` - дата создания (Long)

#### Visit (Визит)
* ```id``` - идентификатор (Long, Primary Key)
* ```patientId``` - внешний ключ на Patient (Long, Foreign Key)
* ```date``` - дата и время визита (Long)
* ```doctorName =``` - имя врача (String, nullable)
* ```diagnosis``` - диагноз (String, nullable)
* ```createdAt``` - дата создания (Long) 

### Связи
* __One-to-Many:__ Один пациент - много записей
* __CASCADE DELETE:__ При удалении пациента автоматически удаляются все его визиты
